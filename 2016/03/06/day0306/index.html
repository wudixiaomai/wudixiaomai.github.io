  • 防盗链

String referer=request.getHeader("Referer");
If(referer==null||!referer.startWith("http://localhost")){
response.sendRedirect("/day06/index.jsp");

}
  • 各种表单输入项数据的获取
    1.text、password、radio、checkbox、
    2.file、select、textarea、 hidden、
    3.image、button给js编程用

  • 请求参数的中文乱码问题

    当表单的method=”post”,用以下处理。


      request.setCharacterEncoding(“UTF-8”);//要加上这一句，设置响应体中文字符编码
      String username = request.getParameter(“username”);
      //或者改下tomcat的配置

    当表单的method=”get”,用以下处理。

      new  String(username.getBytes(“ISO-8859-1”), “UTF-8”);
      //或者改下tomcat配置 URIEncoding，不建议使用

    以上为方立勋说的方式。

    自己查了下官网的文档有关tomcat的配置说明，tomcat8默认以utf-8解码如这种URI中带着这种编码%E9%99%88%E7%AE%80%E5%8D%95（陈简单）编码的请求数据。

    结论：当get请求时，URI带着这种
    tomcat8默认用的是utf-8。而使用post时，一定要request.setCharacterEncoding(“UTF-8”);
